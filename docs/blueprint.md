# **App Name**: MetaMe Profile Forge

## Core Features:

- AI Profile Generation: AI-powered profile generation tool. Generates personalized profile content based on user prompts.
- Profile Customization: User profile customization. Allows users to edit and refine AI-generated content with personal details, interests, and skills.
- Image Upload: Profile Image Upload. Allows users to upload and display a profile photo.
- Profile Preview: Profile Preview. Offers a real-time preview of the completed profile.
- PDF Download: PDF Export. Enables users to download their profile as a PDF document.

## Style Guidelines:

- Primary color: Soft blue (#72A0C1), a versatile, calm hue which does not call too much attention to itself.
- Background color: Very light desaturated blue (#F0F4F7).
- Accent color: Light vibrant blue (#9DC3E6) for highlights and CTAs.
- Headline font: 'Poppins', a geometric sans-serif.
- Body font: 'Inter', a grotesque-style sans-serif. Use 'Poppins' for headlines, and 'Inter' for longer body text.
- Simple line icons to represent profile sections and actions.
- Clean, single-column layout for easy navigation and readability.
